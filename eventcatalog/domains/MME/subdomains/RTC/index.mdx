## Overview

The Real-Time Comms subdomain manages WebSocket connections and real-time messaging between users and services.
It supports event broadcasting (e.g., game state updates, chat), presence tracking, and pub/sub mechanics.
This layer ensures that all connected clients receive timely and consistent updates, maintaining the responsiveness required for live gameplay or collaboration.
