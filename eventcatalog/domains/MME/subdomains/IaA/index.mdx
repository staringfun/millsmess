## Overview

The Identity & Analytics subdomain owns the user model, session authentication, and activity tracking.
It ensures secure access to the system via token-based auth and manages user accounts and profiles.
It also collects data on sessions, user engagement, and gameplay metrics for monitoring and insights.
This data can be used for dashboards, reporting, or matchmaking logic.
